## âš™ï¸ How It Works

1. The soil moisture sensor continuously reads moisture levels.
2. If moisture falls **below threshold**, the **relay turns ON** and activates the water pump.
3. Once the soil becomes sufficiently moist, the **relay turns OFF**, stopping the water flow.
4. This cycle helps maintain ideal soil conditions automatically.
